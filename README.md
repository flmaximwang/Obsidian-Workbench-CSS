# Maxim's Workbench

This is a repository for an Obsidian theme.

## Features

- Floating sidebars
- csslasses for print

## Development

- Use `npm run build:css` to generate theme.css

## Print

- Add `a4-page` to cssclasses frontmatter to make sure your note will be exported to PDF that is compatible with A4 pages.
- IMPORTANT: Switch to reading view, scroll down all your notes before print.
